Eventually

Test Page

Eventually allows you to use addEvent or removeEvent functions to take advantage of addEventListener or removeEventListener and not worry about old browsers.

Here you have the new functions you'll be using to achieve this:

  • elem.addEvent("event", function)
  • elem.removeEvent("event", function)
  • event: here goes the event indeed.
  • function: and here goes the function you want to add.

Try it up and let me know if there are any bugs!
