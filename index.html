Eventually

Eventually is a small fallback for browsers like IE8 that don't support addEventListener or removeEventListener.

Here you have the new functions you'll be using to achieve this:

  • elem.addEvent("event", function)
  • elem.removeEvent("event", function)
  • event: here goes the type of event.
  • function: and here goes the function name you want to add.

You can also send an anonymous function as an argument.

Here you have three buttons. The first one adds an event, the second one removes it and the third triggers it.
