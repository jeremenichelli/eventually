Test Page

This is a page to test Eventually, a script that allows you to use addEvent or removeEvent functions to take advantage of addEventListener or removeEventListener and not worry about old browsers.

Here you have the new functions you'll be using to achieve this:

  • addEvent("queryString", "event", function)
  • removeEvent("queryString", "event", function)
  • queryString: here goes your selector like "a.btn" or ".call-to-action"
  • event: here goes the event indeed.
  • function: and here goes the function you want to add.

Try it up and let me know if there are any bugs!
